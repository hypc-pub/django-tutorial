Django安装

安装Python

Linux系统下安装

linux系统已经自带Python，可以直接使用。

Windows系统下安装

直接去 https://www.python.org/downloads/ 下载一个最新稳定版，安装即可。

安装Django

直接使用pip install Django命令安装， Windows系统下可以使用python -m pip install Django命令安装。
